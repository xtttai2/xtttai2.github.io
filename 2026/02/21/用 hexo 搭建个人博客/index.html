用 hexo 搭建个人博客

准备

  • Node.js
  • Git
  • gihub 账号

配置 SSH

  1. 运行 ssh-keygen -t rsa -C "Github 注册的邮箱地址"

  2. 不停回车或肯定，直到出现一个奇怪图案。

  3. 复制 ~\.ssh\id_rsa.pub 文件里面的所有内容。

  4. 登录到 github，点击右上角头像->“Settings”->“SSH and GPG keys”->“New SSH key”。

  5. “Title” 随便写，在”Key“里粘贴你之前复制的内容。

  6. 点击“Add SSH Key”。

  7. 终端输入 ssh -T git@github.com，出现“Hi…”的信息就大功告成了。

部署

创建 github 仓库

  1. 进入 github，点击右上角“+”->“New repository”。

  2. “Repository name”写 你的用户名.github.io。

  3. “Add README”选择“On”。

  4. 其他选项不用改，点击“Create repository”。
